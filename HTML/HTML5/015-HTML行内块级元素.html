这是一个div

媒体元素 audio video
文档结构元素 header footer nav article section
以上标签属于块级元素。块级元素有以下特点：

  1. 独占一行或一块区域
  2. 宽度默认是100%
  3. 可以设置宽度和高度
  4. 可以包含内联元素和其他块级元素
  5. 默认情况下不允许嵌套
  6. 可设置盒模型属性

这是一个span

文本相关元素 span a
强调和语义元素 strong em
图片相关元素 img（行内块元素）
表单元素 input label
文本标记元素 abbr cite code br time
引用元素 q
变量元素 var
分隔元素 br
嵌套文本元素 b i
以上标签属于行内元素。行内元素有以下特点：

  1. 在同一行显示
  2. 宽度和高度受内容影响 默认不接受宽度和高度设置
  3. 不会开始新的一行
  4. 可嵌套在块级元素内
  5. 水平方向的边距、填充有效，垂直方向无效
  6. 一般不能包含块级元素
